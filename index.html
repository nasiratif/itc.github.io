About the Software

Islāmic Text Copier (ITC abbreviated) is an open-source, simple but very useful PC program that allows you to easily copy islamic expressions in Arabic. You can either click the image of the text to copy it, or use hotkeys such as Alt + 1 for ﷻ, Alt + 2 for سبحانه و تعالى, etc. This is especially useful when writing articles.

Originally, I wrote it just for myself to use to make writing articles easier, but decided to publish it online as it would be useful for others as well.

IMPORTANT: Please note that Islāmic Text Copier is not a code-signed app, and to get this done is too expensive for me, so because of this, it might be falsely detected as a virus. You may need to add the software to your antivirus's Exclusions folder to bypass this. If you get a SmartScreen popup (which you most likely will), click More info and select Run anyway at the bottom to start the program. By Allāh, this is not genuinely harmful software.
